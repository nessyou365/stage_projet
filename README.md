# stage_projet
# 🚀 Intelligent Recruitment System (Web & Mobile)

## 📌 Description

AI Recruitment System est une plateforme intelligente de gestion du recrutement qui combine une application web (dashboard RH) et une application mobile (candidat).

Le système permet d'automatiser le processus de recrutement grâce à l'intelligence artificielle, en analysant les CV, en évaluant les candidats et en aidant les recruteurs à prendre des décisions efficaces.

---

## 🎯 Objectifs

* Automatiser le tri des CV
* Améliorer la prise de décision RH
* Faciliter la gestion des candidats
* Intégrer l'intelligence artificielle dans le recrutement
* Permettre les candidatures spontanées

---

## 🧠 Fonctionnalités principales

### 👨‍💼 Admin (Web App)

* 📊 Dashboard avec KPI et statistiques
* 📄 Gestion des offres d’emploi
* 👥 Gestion des candidats
* 🤖 Analyse IA des CV (matching, skill gap)
* 🎯 Ranking des candidats
* 🎤 Interview automatisée (IA)
* 🔄 Pipeline de recrutement (Kanban)
* 🧾 Prise de décision (Accept / Reject)
* 📝 Ajout de notes
* 📂 Gestion des candidatures spontanées
* 💡 Recommandations intelligentes

---

### 👤 Candidat (Mobile App)

* 📤 Upload CV
* 📝 Modifier profil
* 📩 Suivi des candidatures
* 🔔 Notifications en temps réel
* 🧠 Feedback IA (score, suggestions)
* 🎤 Participation à l’interview IA
* 🚀 Candidature spontanée

---

## 🏗️ Architecture

* 📱 Mobile App (Flutter)
* 🌐 Web App (React)
* ⚙️ Backend API (Laravel / Node.js)
* 🗄️ Base de données (MySQL)
* 🤖 AI (OpenAI API)
* 🔔 Notifications (Firebase)

---

## 🔄 Workflow

1. Le candidat envoie son CV
2. Le backend analyse les données
3. L’IA génère un score et des recommandations
4. Le recruteur visualise les résultats
5. Une décision est prise (Accept / Reject)
6. Le candidat reçoit une notification

---

## 🧰 Technologies

* Frontend Web: React + Tailwind CSS
* Mobile: Flutter
* Backend: Laravel / Node.js
* Database: MySQL
* AI: OpenAI API
* Notifications: Firebase

## 🚀 Installation

### 1. Cloner le projet

```bash
git clone https://github.com/nessyou365/stage_projet.git
```

### 2. Backend

```bash
cd backend
composer install
php artisan serve
```

### 3. Frontend

```bash
cd frontend-web
npm install
npm run dev
```

### 4. Mobile

```bash
cd mobile-app
flutter pub get
flutter run
```
