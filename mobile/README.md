# HireQ Mobile

Application mobile candidat pour consulter les offres, envoyer des candidatures et suivre leur statut.

## Stack

- Flutter
- Firebase Auth
- Cloud Firestore
- File picker
- Syncfusion PDF

## Installation

```bash
flutter pub get
flutter run
```

## Structure

```text
mobile/
├── lib/        # Code Flutter principal
├── assets/     # Images utilisees par l'application
├── test/       # Tests Flutter
├── android/    # Projet Android
├── ios/        # Projet iOS
├── web/        # Build target web Flutter
├── windows/    # Build target Windows
├── macos/      # Build target macOS
└── linux/      # Build target Linux
```
